== Description ==

Improvements for user management – adds 3 new fields to users list for each user. Each field is automatically updated when the action is taken place.

Also features a possibility to send email to all users.

**14 days trial available for Pro version, no credit card required.**

For more information please visit [usermagicpro.com](https://www.usermagicpro.com).

= New fields for each user: =
* Last login timestamp
* Number of total logins
* Last password change date and time

You may also send email to all users in your WordPress user database or to a specific group of users.

= Features in Free version: =
* Timestamp for last login
* Counter for user logins
* Last password change date and time

= Features in Pro version: =
* Send email to users **using Mailgun**

Contact author [here](https://anssilaitila.fi/).

== Installation ==

1. Activate the plugin from WordPress plugin directory or manually upload it to your site
